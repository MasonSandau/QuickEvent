# User Endpoints

## Login
- **Route:** `/login`
- **Method:** `POST`
- **Description:** Authenticates a user and creates a session.

## Register
- **Route:** `/register`
- **Method:** `POST`
- **Description:** Registers a new user.

## Logout
- **Route:** `/logout`
- **Method:** `GET`
- **Description:** Logs out the current user.
